# v2ray-sub

Auto-collected and filtered V2Ray / Hysteria2 node subscription.

## Usage

Subscription URL:

```
https://raw.githubusercontent.com/Achillesy/v2ray-sub/main/sub.txt
```

Add this URL to your v2rayN (or compatible client) subscription list and update.
